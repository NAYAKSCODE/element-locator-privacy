Privacy Policy – Element Locator

Last updated: 2025-01-01

Element Locator is a Chrome extension that helps users capture element selectors (e.g., XPath) from the page they are viewing.

Data collection

We do not collect, transmit, sell, or share any personal data. The extension does not send data to any server.

Local storage

Selectors captured by the user (XPath, tag name, optional text, and a timestamp) are stored locally in the browser via chrome.storage.local. Users can delete items individually or clear all at any time.

Permissions use

  • activeTab: Run on the tab the user activates to compute a selector.
  • scripting: Inject the packaged content script to highlight elements and capture selectors.
  • storage: Save user‑initiated selectors locally.
  • tabs: Identify and message the active tab while capture is enabled.
  • Host permissions: Operate on pages the user chooses during capture. No content is uploaded.

Remote code

No remote code is used. All code ships with the extension; there is no use of external scripts, eval, or remote Wasm.

Third parties & analytics

No third‑party analytics, ads, or trackers.

Children’s privacy

The extension is not directed to children under 13.

Changes

We may update this policy and will update the “Last updated” date above.
